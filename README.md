# Mock Interviewer Skill

一个为 [Claude Code](https://claude.ai/code) 设计的严苛面试官模拟器，专攻头部大厂 AI Lab / 大模型应用方向的面试场景。

## 功能

**两种输入方式**
- 提供简历文件路径（PDF / TXT / MD），自动读取解析
- 直接粘贴个人信息、项目经历、技能栈

**四种输出模式**

| 模式 | 说明 |
|------|------|
| `[1]` 仅列面试问题 | 根据简历生成全流程定制化问题，无答案 |
| `[2]` 问题 + 回答大纲 | 每题附 2-4 个要点提示 |
| `[3]` 问题 + 完整参考回答 | 含工程深度回答 + 面试官视角点评 |
| `[4]` 模拟真实面试 | 逐轮对话，我问你答，动态追问 |

**面试流程（Phase 1→8）**
1. 教育背景与学习能力摸底
2. 经历深挖（个人贡献边界、指标真实性、技术细节）
3. 场景题与系统设计（从简历技术栈动态派生，非固定模板）
4. 交叉面提问（模拟非本方向面试官）
5. AI 认知与工程素养
6. 公司与岗位匹配
7. 手撕代码
8. 反问环节

**岗位感知**：自动识别目标岗位类型（AI 应用工程师 / 模型工程师 / 算法研究员），调整考察重心。

## 安装

将 `SKILL.md` 放入 Claude Code 的 skills 目录：

```
~/.claude/skills/Mock-Interviewer-Skill/SKILL.md
```

## 使用

在 Claude Code 中调用：

```
/mock-interviewer
```

然后提供简历路径或直接粘贴信息，选择模式即可。

## 风格

- 不寒暄，不鼓励，不给正向反馈
- 对"团队项目"天然怀疑，对"漂亮指标"条件反射式追问
- 所有问题从简历内容派生，禁止泛化套题
- 场景题动态生成，不使用固定示例，避免对模型产生副作用
